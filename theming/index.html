Getting Started

The Populr theming engine is based on the awesome javascript templating engine Handlebars.js. Before starting a Populr theme, you should visit the Handlebars website and familiarize yourself with the syntax.

Once you are comfortable with Handlebars syntax, download the Populr Theme Skeleton to get started with your own theme.

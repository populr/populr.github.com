Creating Populr Themes

  • Getting Started
  • Anatomy of a Theme

Creating your own Populr theme is easy, once you get to know the system. This guide will help you learn the fundamentals of theming.

Heads up! The Populr theming engine is based on the javascript templating system Handlebars.js. Before starting a Populr theme, you should visit the Handlebars website and familiarize yourself with the syntax. Seriously, if you don't learn how to use handlebars, you won't learn how to theme Populr. We promise.

Once you are comfortable with Handlebars syntax, download the Populr Theme Skeleton to get started with your own theme.

Anatomy of a Theme

A Populr theme is divided up into two main sections:

Regions

At it's core, a region is simply a container for assets. The magic happens when defining what type of layout engine a region uses to display those assets. We'll talk more about this in a bit. The two layout engines currently available are:

  • Columnizer
  • Isotope

Assets

Assets are the heart of Populr. Each item of content in a Pop is one of five asset types:

  • Text
  • Image(s)
  • Recorded Video(s)
  • Document(s)
  • Embed(s)

Each asset in a Pop is rendered according to the region it resides in as well as the size selected for the asset.
